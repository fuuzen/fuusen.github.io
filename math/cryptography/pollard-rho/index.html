# 大数字符串输入输出处理

本次实验不像先前的实验都是二进制输入输出，而是十进制字符串输入输出。

对于输出，由于输出的离散对数是有限域 nnn 以内的，而 nnn 在本次实验限定为 64 位以内，所以可以直接使用 C++ 本身自带的 64 位数输出。

# Pollard-Rho

Pollard-Rho 的算法我也采用了面向对象的设计，将一个 DLP 问题的条件作为成员变量封装在里面：

  • ppp 定义在 Field f ，同时也是整个 Pollard-Rho 算法大数运算时所基于的有限域。
  • uint64_t n 即为 nnn，因为输入限制在 64 位以内，所以不必使用大数。
  • Int alpha = Int(f) 即为 α\alphaα，属于有限域 ppp。
  • Int beta = Int(f) 即为 β\betaβ，属于有限域 ppp。
